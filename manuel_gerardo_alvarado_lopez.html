Examen Git y Github

Alumno: Manuel Gerardo Alvarado López

  1. Que es un sistema de control de versiones?

    Es un mecanismo para manejar los cambios en archivos de manera que pueda trabajarse en conjunto y que permite un control sobre los avances y entregas del proyecto.

  2. Cual es la diferencia entre git y github?

    Github hace uso de la tecnología de Git, Git es el sistema manejador de versiones y Github es un repositorio remoto que utiliza git.

  3. Cómo creo una rama en git?

    git branch -M [Nombre de la Rama Nueva]

  4. Cómo clono un repositorio?

    Te situas sobre el directorio local donde quieras clonar tu repositorio remoto escribes el comando $ git clone https://github.com/Usuario/nombreDelRepo
